## 👨‍💼 Admin User Guide

### 1️⃣ Adding a New Room

1. Click **"Room Management"** in the sidebar
2. Click the **"+ Add New Room"** button (top right)
3. Fill in the room details:
   - Room Number (e.g., 401)
   - Select Floor (1-5)
   - Choose Type (AC or Non-AC)
   - Select Capacity (1-6 persons)
   - Enter Price per Night
   - Set Status (Vacant/Booked/Maintenance)
   - Add Amenities (comma-separated: WiFi, TV, Bathroom)
4. Click **"Add Room"**
5. ✅ Success! Room is now available in the system

### 2️⃣ Editing a Room

1. Go to **"Room Management"**
2. Find the room you want to edit
3. Click the **"Edit"** button on the room card
4. Update the desired information
5. Click **"Update Room"**
6. ✅ Changes saved!

### 3️⃣ Deleting a Room

1. Go to **"Room Management"**
2. Find the room to delete
3. Click the **trash icon** button
4. Confirm the deletion in the popup
5. ✅ Room removed from system

### 4️⃣ Managing Bookings

**View All Bookings:**
1. Click **"All Bookings"** in sidebar
2. See statistics: Total, Confirmed, Pending, Completed
3. Use search bar to find specific bookings
4. Filter by status using the filter buttons

**Mark Booking as Completed:**
1. Find the booking in the list
2. Click the **green checkmark** icon
3. Confirm the action
4. ✅ Booking marked complete & room becomes vacant

**Cancel a Booking:**
1. Find the booking in the list
2. Click the **red X** icon
3. Confirm cancellation
4. ✅ Booking cancelled & room becomes vacant

**View Booking Details:**
1. Click the **eye icon** on any booking
2. See complete guest information
3. View check-in/out dates and times
4. See purpose of visit

### 5️⃣ Managing Employees

**Add New Employee:**
1. Click **"Employees"** in sidebar
2. Click **"+ Add Employee"** button
3. Fill in employee details:
   - Full Name
   - Email Address (@simats.edu)
   - Phone Number
   - Department
   - Role (Employee or Administrator)
4. Click **"Add Employee"**
5. ✅ Employee account created!

**Edit Employee:**
1. Find the employee card
2. Click **"Edit"** button
3. Update information
4. Click **"Update Employee"**

**Remove Employee:**
1. Find the employee
2. Click the **trash icon**
3. Confirm removal
4. ✅ Employee removed

## 👨‍💻 Employee User Guide

### 1️⃣ Searching Available Rooms

1. From **Dashboard**, see all available rooms
2. Use the **search bar** to find specific rooms
3. View room details:
   - Room number and floor
   - AC/Non-AC type
   - Capacity
   - Price per night
   - Amenities

### 2️⃣ Creating a Booking

1. Find the desired vacant room
2. Click **"Book Now"** button
3. Fill in guest information:
   - **Guest Name** (required)
   - **Phone Number** (required, format: +91 XXXXX XXXXX)
   - **Email Address** (required)
   - **Check-in Date** (select from calendar)
   - **Check-in Time** (select time)
   - **Check-out Date** (must be after check-in)
   - **Check-out Time** (select time)
   - **Number of Guests** (cannot exceed room capacity)
   - **Purpose of Visit** (required - Conference, Guest Lecture, etc.)
4. Click **"Confirm Booking"**
5. ✅ Booking confirmed! Room status updated to "Booked"

### 3️⃣ Viewing Your Bookings

1. Click **"My Bookings"** in sidebar
2. See your booking statistics:
   - Total Bookings
   - Active Bookings
   - Upcoming Bookings
3. Search for specific bookings
4. Click **"View Details"** on any booking to see complete information

## 🆘 Common Issues

**Can't delete a room?**
- Ensure there are no active bookings for that room

**Booking form shows error?**
- Check all required fields are filled
- Verify email format is correct
- Ensure check-out is after check-in
- Verify guest count doesn't exceed capacity

**Can't see a booking?**
- Try clearing search filters
- Check the correct status filter is selected
- Refresh the page
